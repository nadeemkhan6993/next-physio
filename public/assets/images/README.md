# Image Assets

Place your physiotherapy/healthcare images in this folder.

## Instructions:
1. Add your images here (e.g., `physio1.jpg`, `physio2.jpg`, etc.)
2. Update the image URLs in login/signup pages to reference these files:
   - Instead of: `https://external-url.com/image.jpg`
   - Use: `/assets/images/physio1.jpg`

## Supported formats:
- JPG/JPEG
- PNG
- WebP
- GIF

## Example:
If you place an image named `patient-care.jpg` in this folder, reference it as:
```tsx
<img src="/assets/images/patient-care.jpg" alt="Patient care" />
```
